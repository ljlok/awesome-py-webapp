###用到的工具/库
+ 数据库：mysql
+ 前端:   
         * UIkit : 前端框架 
         * vue.js : MVVM 类似于angular.js 
         * sticky.js : Sticky is a jQuery plugin that gives you the ability to
                       make any element on your page always stay visible. 
         * jinja2 : 一个python模版引擎

+ python库: 
            * watchdog - 接受文件变化通知，在开发时当有文件更新时重启服务，自动更新app
            * subprocess - 创建子进程
            * doctest - 一个测试模块, 利用注释中命令行语法的代码测试
            * functools - functools.wraps(wrapped)用于decorator, 保证在调用经过装饰的函数时，
                          保留原有的__name__,__doc__,而不是返回decrator的信息
            * logging - for log
